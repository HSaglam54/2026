# Schnellstartanleitung

## Installation und erster Start in 15 Minuten

### Schritt 1: Repository Klonen

Öffnen Sie Ihr Terminal und klonen Sie das Repository:
```bash
git clone https://github.com/HSaglam54/2026.git
cd 2026
```

### Schritt 2: Abhängigkeiten installieren

Installieren Sie die benötigten Abhängigkeiten:
```bash
npm install
```

### Schritt 3: Anwendung starten

Starten Sie die Anwendung mit:
```bash
npm start
```

### Schritt 4: Auf die Anwendung zugreifen

Öffnen Sie einen Webbrowser und gehen Sie zu:
```
http://localhost:3000
```

### Schritt 5: Erster Test

Überprüfen Sie, ob alles ordnungsgemäß funktioniert, indem Sie die Startseite aufrufen.
