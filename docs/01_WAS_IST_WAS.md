# Technologien, die verwendet werden

## HTML
HTML (HyperText Markup Language) ist die Standardauszeichnungssprache für Dokumente, die im Web angezeigt werden. HTML beschreibt die Struktur von Webinhalten, indem es verschiedene Elemente zur Verfügung stellt, um Texte, Links, Bilder usw. darzustellen.

## CSS
CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache, die verwendet wird, um das Layout und das Aussehen von HTML-Dokumenten zu gestalten. Mit CSS können Sie Farben, Schriftarten, Abstände und Layouts definieren und gestalten.

## JavaScript
JavaScript ist eine Programmiersprache, die häufig in der Webentwicklung verwendet wird. Sie ermöglicht es Entwicklern, interaktive Elemente auf Webseiten zu erstellen, von einfachen Animationen bis hin zu komplexen Anwendungen.

## Node.js
Node.js ist eine serverseitige JavaScript-Laufzeitumgebung, die es Entwicklern ermöglicht, serverseitige Anwendungen mit JavaScript zu erstellen. Es ermöglicht die Erstellung von skalierbaren Netzwerk-Anwendungen.

## React
React ist eine JavaScript-Bibliothek zur Erstellung von Benutzeroberflächen. Sie ermöglicht die Erstellung von wiederverwendbaren UI-Komponenten und hilft Entwicklern, große Anwendungen mit komplexen Benutzerinteraktionen zu verwalten.

## SQL
SQL (Structured Query Language) ist die standardisierte Sprache zum Verwalten und Abfragen von Datenbanken. SQL wird verwendet, um Daten zu speichern, zu aktualisieren und zu löschen, sowie um Informationen aus Datenbanken abzurufen.

## Git
Git ist ein Versionskontrollsystem, das Entwicklern hilft, Änderungen am Code nachzuvollziehen und zusammenzuarbeiten. Es ermöglicht das Verwalten von verschiedenen Versionen von Projektdateien und das Arbeiten an Projekten mit mehreren Entwicklern.

## Docker
Docker ist eine Plattform zur Automatisierung der Bereitstellung von Anwendungen in Containern. Container sind leichtgewichtige, portable Einheiten, die eine Anwendung und alle deren Abhängigkeiten beinhalten, was die Bereitstellung und den Betrieb erheblich vereinfacht.
