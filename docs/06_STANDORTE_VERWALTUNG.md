# Standorte Verwaltung

Um Standorte hinzuzufügen und zu verwalten, folgen Sie diesen Schritten:

## 1. Standort hinzufügen
1. Gehen Sie zum Abschnitt "Standorte" in Ihrer Anwendung.
2. Klicken Sie auf die Schaltfläche "Neuen Standort hinzufügen".
3. Geben Sie die erforderlichen Informationen ein:
   - Name des Standorts
   - Adresse
   - Kontaktinformationen
   - Weitere relevante Details
4. Klicken Sie auf "Speichern", um den Standort hinzuzufügen.

## 2. Standorte bearbeiten
1. Um einen bestehenden Standort zu bearbeiten, gehen Sie erneut zu "Standorte".
2. Wählen Sie den Standort aus, den Sie ändern möchten.
3. Aktualisieren Sie die Informationen und klicken Sie auf "Änderungen speichern".

## 3. Standort löschen
1. Wählen Sie den Standort aus, den Sie löschen möchten.
2. Klicken Sie auf die Schaltfläche „Löschen“ und bestätigen Sie die Aktion.

## 4. Standortdetails anzeigen
- Wenn Sie die Details eines Standorts anzeigen möchten, wählen Sie ihn einfach aus der Liste aus. Die Informationen werden im Detailbereich angezeigt.

## Tipps
- Stellen Sie sicher, dass die eingegebenen Informationen genau sind, um Verwirrung zu vermeiden.
- Verwenden Sie die Suchfunktion, um schnell auf einen bestimmten Standort zuzugreifen.
